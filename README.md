## 연구 참여 이력

### 연구실 내부 연구: RTiK (Real-Time implant Kernel)
- Windows 10을 위한 실시간 커널 포팅 및 성능개선 
- ARM 리눅스를 위한 실시간커널 포팅 및 개발 
- 안드로이드 시스템을 위한 실시간 커널 설계 및 개발 
- 범용 리눅스를 위한 실시간 커널의 타이머 성능 개선 
- 임베디드용 실시간 커널의 현대화 및 포팅 (Cortex-M4F)

### 실시간 시스템 / 운영체제 / 스케줄링
- (2017~2019) 멀티프로세서 IoT 센싱 디바이스의 수행성능 강화를 위한 스케줄링 방안 연구
- (2016) 이기종 노드를 이용한 대용량 FDTD 시뮬레이션 연산 환경 구축 및 성능 분석 
- (2015) ARM용 시스템에서의 실시간 커널 확장 
- (2014) 임의의 주기(Hz 기반)를 가지는 테스크를 위한 스케줄링 방안 연구

### 보안 운영체제 관련 연구.
- (2019) 임베디드 시스템을 위한 시큐어 부팅 기술 안전성 강화방안 연구 
- (2018) Secure boot 기법의 임베디드 시스템 적용에 관한 연구 
- (2017) Trusted Computing을 위한 경량 보안 운영체제 기반 기술 연구 
- (2016) Secure OS가 적용된 리눅스 시스템 활용방안 연구

### 디바이스 드라이버

- (2018) 딥러닝 파라미터 저장을 위한 메모리박스 시스템 고도화
